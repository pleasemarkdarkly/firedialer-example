# Slack Integration

Generally, create app here `https://api.slack.com/`. Generate OAUTH token.

1. Add Incoming Webhook
2. Add Slash Commands and corresponding Request URL

## OAUTH Permissions
Add the following scopes:
* chat:write
* commands
* incoming-webhook
